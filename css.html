CSS

Hojas de estilo en cascada para controlar la presentación visual de un documento web.

Introducción

Lenguaje de estilos: apariencia, layout y diseño responsivo.

El CSS (Cascading Style Sheets) es el lenguaje utilizado para definir la apariencia visual de los elementos HTML. Fue propuesto por Håkon Wium Lie en 1994 y estandarizado por el W3C en 1996 como CSS1.

Posteriormente, CSS2 añadió posicionamiento avanzado, y CSS3 se organizó en módulos independientes que incorporan Flexbox, Grid Layout, animaciones y transiciones.

CSS es fundamental para construir interfaces modernas, accesibles y responsivas.

Usos principales

  • Diseño visual de páginas web
  • Layouts modernos (Flexbox / Grid)
  • Adaptación responsiva
  • Animaciones y transiciones
  • Sistemas de diseño
