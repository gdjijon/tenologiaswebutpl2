JavaScript

Lenguaje para comportamiento: eventos, lógica y experiencias dinámicas en la web.

Introducción

Lenguaje de programación del navegador: interactividad, DOM y evolución ECMAScript.

JavaScript fue creado en 1995 por Brendan Eich en Netscape con el objetivo de aportar dinamismo a las páginas web. Inicialmente llamado LiveScript, fue renombrado para alinearse comercialmente con Java.

Su estandarización recayó en ECMA International bajo el estándar ECMAScript, cuya primera versión apareció en 1997.

A partir de ES6 (2015), JavaScript evolucionó significativamente con clases, módulos, promesas, async/await y actualizaciones anuales.

Usos modernos

  • Manipulación del DOM y eventos
  • Interfaces interactivas
  • Aplicaciones web (SPA)
  • Backend con Node.js
  • Apps móviles y de escritorio
